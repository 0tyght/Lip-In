# คู่มือใช้งานจริง Lip In Money

แอปนี้ออกแบบให้ใช้กับเงินจริงและธนาคารไทยเท่านั้น ไม่มี Plaid, Open Banking ต่างประเทศ, Worker backend หรือการเก็บ bank secret

## 1. เปิดใช้งานบน iPhone

ใช้ URL:

```text
https://0tyght.github.io/Lip-In/
```

ติดตั้งเป็น PWA:

1. เปิด Safari บน iPhone
2. เข้า URL ด้านบน
3. กด Share
4. เลือก `Add to Home Screen`
5. เปิดจากไอคอน `Lip In`

ข้อมูลหลักเก็บในเครื่องผ่าน localStorage ควรสำรอง JSON เป็นระยะ

## 2. เริ่มตั้งค่าข้อมูลจริง

แอปเริ่มว่าง มีแค่:

- `บัญชีหลัก` ยอด 0 บาท
- `เงินสด` ยอด 0 บาท

ให้ทำก่อนใช้จริง:

1. ไปที่ `กระเป๋า`
2. แก้ชื่อกระเป๋าให้ตรงกับบัญชีไทยที่ใช้ เช่น `KBank`, `SCB`, `Krungthai`, `Krungsri`, `เงินสด`
3. ใส่ยอดคงเหลือจริง ณ วันที่เริ่มใช้
4. เพิ่มกระเป๋าหนี้สินถ้ามีบัตรเครดิตหรือรายการผ่อน

## 3. อ่านสลิปโอนเงินจากเครื่อง

ไปที่ `เมนู` หรือหน้า `กระเป๋า` แล้วกด `สลิปธนาคารไทย`

จากนั้น:

1. กด `เลือกสลิปในเครื่อง`
2. เลือกรูปสลิปได้หลายรูป
3. แอปจะพยายามอ่าน QR ในสลิปบนเครื่องทันที
4. ถ้าอ่านยอด/เลขอ้างอิงได้ แอปจะเติมข้อมูลตั้งต้นให้
5. ถ้ายังไม่รู้ว่ารายการคืออะไร แอปจะส่งเข้า `To do สลิป`

ข้อจำกัดที่ต้องรู้:

- รูปไม่ถูกส่งออกจากเครื่อง
- ถ้า browser บนเครื่องไม่รองรับการอ่าน QR หรือสลิปไม่มี QR ที่อ่านได้ รายการจะเข้า To do ให้กรอกเอง
- การอ่านข้อความ OCR จากภาพทั้งใบยังไม่ได้ใช้ cloud OCR เพื่อรักษาความเป็นส่วนตัว

## 4. ทำงานกับ To do สลิป

เมื่อมีสลิปรอตรวจ จะเห็น `To do สลิป` ในหน้าแรกและในหน้าสลิป

ขั้นตอน:

1. กด `บันทึกรายการ`
2. ตรวจยอด วันที่ และเลขอ้างอิง
3. ใส่ชื่อรายการ เช่น `โอนค่าอาหาร`, `รับเงินจากลูกค้า`, `เติมเงิน`
4. เลือกประเภท `รายจ่าย`, `รายรับ`, หรือ `โอนเงิน`
5. เลือกหมวดและกระเป๋า
6. กดบันทึก

หลังบันทึกสำเร็จ สลิปนั้นจะถูก mark ว่าเสร็จและออกจาก To do

## 5. นำเข้า Statement CSV ธนาคารไทย

ใช้เมื่อมีไฟล์ statement จากแอป/เว็บธนาคาร:

1. เปิด `สลิปธนาคารไทย`
2. กด `นำเข้า CSV ธนาคารไทย`
3. เลือกไฟล์ CSV

คอลัมน์ที่รองรับ:

- วันที่: `date`, `transaction date`, `วันที่`, `วันทำรายการ`
- เวลา: `time`, `เวลา`
- รายละเอียด: `description`, `details`, `รายการ`, `รายละเอียด`
- เงินออก: `debit`, `withdrawal`, `เงินออก`, `ถอน`, `จ่าย`
- เงินเข้า: `credit`, `deposit`, `เงินเข้า`, `ฝาก`, `รับ`
- จำนวนเงิน: `amount`, `transaction amount`, `จำนวนเงิน`, `ยอดเงิน`
- เลขอ้างอิง: `reference`, `transaction id`, `เลขที่รายการ`, `อ้างอิง`

แอปกันรายการซ้ำด้วยเลขอ้างอิง หรือ hash จากวันที่ เวลา รายละเอียด จำนวนเงิน และบัญชี

## 6. บันทึกรายการเอง

กดปุ่ม `+` แล้วเลือก:

- `รายจ่าย`
- `รายรับ`
- `โอนเงิน`
- `ชำระ`

ถ้ายังไม่แน่ใจ ให้ตั้งสถานะเป็น `รอตรวจสอบ` ก่อน รายการสถานะนี้จะยังไม่กระทบยอดกระเป๋าจนกว่าจะบันทึกเป็น `ลงบัญชีแล้ว`

## 7. ตั้งงบ เป้าหมาย และผ่อนชำระ

- `งบประมาณ`: ตั้งวงเงินต่อหมวด เช่น อาหาร เดินทาง กาแฟ ค่าไฟ
- `เป้าหมาย`: เก็บเงิน เช่น เงินสำรอง ทริป ซื้อของใหญ่
- `ผ่อนชำระ`: ติดตามเงินต้น ดอกเบี้ย และจำนวนงวด

## 8. สำรองข้อมูล

ไปที่ `เมนู`

- `สำรอง JSON`: เก็บข้อมูลทั้งหมดของแอป
- `ส่งออก CSV`: เก็บรายการธุรกรรมไว้เปิดใน Excel/Google Sheets

ควรสำรอง JSON เป็นประจำ โดยเฉพาะก่อนล้างข้อมูล Safari หรือเปลี่ยนเครื่อง

## 9. อัปเดตเวอร์ชัน

PWA จะอัปเดตผ่าน Service Worker เมื่อเปิดแอปและมีเวอร์ชันใหม่ ไม่ต้องติดตั้งใหม่

ถ้าต้องการตรวจเอง:

1. ไปที่ `เมนู`
2. กด `ตรวจเวอร์ชันล่าสุด`
3. ปิดเปิดแอปอีกครั้งหลังอัปเดต

## 10. วิธีใช้ประจำวันที่แนะนำ

ระหว่างวัน:

- เจอสลิปโอนเงิน ให้เปิด `สลิปธนาคารไทย` แล้วเลือกจากรูปในเครื่อง
- สลิปที่ยังไม่รู้หมวด ให้ปล่อยไว้ใน To do ก่อน
- รายการที่จ่ายเงินสดให้กด `+`

สิ้นวัน:

- เปิด `To do สลิป`
- เติมรายละเอียดให้ครบ
- ตรวจยอดกระเป๋า

สิ้นเดือน:

- นำเข้า statement CSV ถ้าธนาคารมีให้ดาวน์โหลด
- ดูรายงานหมวดหมู่และงบประมาณ
- สำรอง JSON
